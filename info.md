# Comfort Band Card

Lovelace card for the [Comfort Band](https://github.com/dpretty/ha-comfort-band) HA integration. Compact tile + expanded modal with Now / Schedule / Profiles / Insights tabs.

**Requires** the `comfort_band` integration ≥ v0.1.0.

```yaml
type: custom:comfort-band-card
zone: gym
```
